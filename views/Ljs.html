Leçon 1 : Introduction au JavaScript

Découvrez JavaScript, le langage de programmation qui donne vie aux pages web !

⚡ Qu'est-ce que JavaScript ?

JavaScript est un langage de programmation qui permet de rendre les pages web interactives. Contrairement au HTML (structure) et CSS (style), JavaScript apporte le comportement et la logique.

Avec JavaScript, vous pouvez :

  • Réagir aux clics et actions de l'utilisateur
  • Modifier le contenu de la page dynamiquement
  • Créer des animations
  • Valider des formulaires
  • Communiquer avec des serveurs (API)

📝 Comment utiliser JavaScript ?

1. JavaScript Inline (dans le HTML)
<button onclick="alert('Bonjour!')">Cliquez-moi</button>

❌ Non recommandé

2. JavaScript Interne (balise <script>)
<body>
    <h1>Ma page</h1>
    
    <script>
        console.log('Hello World!');
    </script>
</body>

✅ Bon pour les tests

3. JavaScript Externe (fichier .js)
<body>
    <h1>Ma page</h1>
    
    <script src="script.js"></script>
</body>

✅✅ Recommandé (meilleure organisation)

🖥️ La console du navigateur

La console est votre meilleur ami pour déboguer. Ouvrez-la avec F12 ou Ctrl+Shift+I (Cmd+Option+I sur Mac).

console.log('Message normal');
console.warn('Avertissement');
console.error('Erreur');
console.table([1, 2, 3]);

👋 Votre premier programme

// Afficher un message
console.log('Hello World!');

// Afficher une alerte
alert('Bienvenue sur mon site!');

// Demander une information
let nom = prompt('Quel est votre nom ?');
console.log('Bonjour ' + nom);
✅ Bonnes pratiques :
  • Placez vos scripts avant la fermeture de </body>
  • Utilisez des fichiers .js externes
  • Commentez votre code : // ou /* */
  • Utilisez la console pour déboguer
  • Terminez chaque instruction par ;

Leçon 2 : Variables & Types de données

Apprenez à stocker et manipuler des données dans vos programmes JavaScript.

📦 Les variables

Une variable est comme une boîte qui contient une valeur. Il existe 3 façons de déclarer une variable :

// var (ancienne méthode, éviter)
var age = 25;

// let (valeur modifiable)
let nom = 'Alice';
nom = 'Bob'; // OK

// const (constante, non modifiable)
const PI = 3.14;
// PI = 3; // ❌ Erreur!
✅ Recommandation :

Utilisez const par défaut, et let seulement si la valeur doit changer. N'utilisez plus var.

🎭 Types de données

// String (texte)
let prenom = "Alice";
let phrase = 'Bonjour tout le monde';
let template = `Bonjour ${prenom}`; // Template literals

// Number (nombres)
let age = 25;
let prix = 19.99;
let negatif = -10;

// Boolean (vrai/faux)
let estMajeur = true;
let estMineur = false;

// Undefined (non défini)
let variable;
console.log(variable); // undefined

// Null (vide volontaire)
let vide = null;

// Array (tableau)
let fruits = ['pomme', 'banane', 'orange'];
console.log(fruits[0]); // 'pomme'

// Object (objet)
let personne = {
    nom: 'Dupont',
    age: 30,
    ville: 'Paris'
};
console.log(personne.nom); // 'Dupont'

🔧 Opérateurs

// Arithmétiques
let a = 10;
let b = 3;
console.log(a + b);  // 13 (addition)
console.log(a - b);  // 7 (soustraction)
console.log(a * b);  // 30 (multiplication)
console.log(a / b);  // 3.333 (division)
console.log(a % b);  // 1 (modulo, reste)
console.log(a ** b); // 1000 (puissance)

// Concaténation
let prenom = 'Alice';
let nom = 'Dupont';
let nomComplet = prenom + ' ' + nom;
// Ou avec template literals
let nomComplet2 = `${prenom} ${nom}`;

// Comparaison
console.log(5 == '5');   // true (valeur égale)
console.log(5 === '5');  // false (valeur et type égaux)
console.log(5 != '5');   // false
console.log(5 !== '5');  // true
console.log(10 > 5);     // true
console.log(10 <= 10);   // true

📊 typeof - Vérifier le type

console.log(typeof 'Alice');    // 'string'
console.log(typeof 42);         // 'number'
console.log(typeof true);       // 'boolean'
console.log(typeof undefined);  // 'undefined'
console.log(typeof null);       // 'object' (bug JS)
console.log(typeof []);         // 'object'
console.log(typeof {});         // 'object'
⚠️ Attention :
  • Utilisez === au lieu de == pour éviter les conversions implicites
  • Les noms de variables sont sensibles à la casse (age ≠ Age)
  • N'utilisez pas de mots réservés (let, if, function, etc.)

Leçon 3 : Les Fonctions

Créez du code réutilisable et organisé avec les fonctions JavaScript.

🎯 Qu'est-ce qu'une fonction ?

Une fonction est un bloc de code réutilisable qui effectue une tâche spécifique. C'est comme une recette de cuisine que vous pouvez suivre plusieurs fois.

📝 Déclaration de fonction

// Fonction simple
function direBonjour() {
    console.log('Bonjour!');
}

// Appeler la fonction
direBonjour(); // Affiche: Bonjour!

// Fonction avec paramètres
function saluer(nom) {
    console.log('Bonjour ' + nom);
}

saluer('Alice'); // Bonjour Alice
saluer('Bob');   // Bonjour Bob

// Fonction avec plusieurs paramètres
function addition(a, b) {
    console.log(a + b);
}

addition(5, 3); // 8

↩️ Return (retourner une valeur)

function additionner(a, b) {
    return a + b;
}

let resultat = additionner(10, 5);
console.log(resultat); // 15

// Fonction avec condition
function estMajeur(age) {
    if (age >= 18) {
        return true;
    } else {
        return false;
    }
}

console.log(estMajeur(20)); // true
console.log(estMajeur(15)); // false

🎨 Function Expression

// Fonction anonyme assignée à une variable
const multiplier = function(a, b) {
    return a * b;
};

console.log(multiplier(4, 5)); // 20

➡️ Arrow Functions (Fonctions fléchées)

// Syntaxe classique
const carre = function(x) {
    return x * x;
};

// Arrow function
const carre2 = (x) => {
    return x * x;
};

// Arrow function simplifiée (1 ligne)
const carre3 = x => x * x;

console.log(carre3(5)); // 25

// Plusieurs paramètres
const addition = (a, b) => a + b;
console.log(addition(3, 7)); // 10

// Sans paramètre
const direHello = () => console.log('Hello');
direHello();

🎁 Paramètres par défaut

function saluer(nom = 'Visiteur', age = 0) {
    console.log(`Bonjour ${nom}, vous avez ${age} ans`);
}

saluer('Alice', 25);  // Bonjour Alice, vous avez 25 ans
saluer('Bob');        // Bonjour Bob, vous avez 0 ans
saluer();             // Bonjour Visiteur, vous avez 0 ans
💡 Exemple pratique :
// Calculatrice simple
function calculer(operation, a, b) {
    if (operation === 'addition') {
        return a + b;
    } else if (operation === 'soustraction') {
        return a - b;
    } else if (operation === 'multiplication') {
        return a * b;
    } else if (operation === 'division') {
        return a / b;
    }
}

console.log(calculer('addition', 10, 5));      // 15
console.log(calculer('multiplication', 4, 3)); // 12
✅ Bonnes pratiques :
  • Donnez des noms descriptifs à vos fonctions
  • Une fonction doit faire une seule chose
  • Utilisez return pour renvoyer une valeur
  • Utilisez les arrow functions pour du code concis

Leçon 4 : Conditions & Boucles

Contrôlez le flux de votre programme avec les structures conditionnelles et les boucles.

🔀 If / Else (Conditions)

let age = 20;

if (age >= 18) {
    console.log('Vous êtes majeur');
} else {
    console.log('Vous êtes mineur');
}

// Avec else if
let note = 15;

if (note >= 16) {
    console.log('Très bien');
} else if (note >= 14) {
    console.log('Bien');
} else if (note >= 12) {
    console.log('Assez bien');
} else if (note >= 10) {
    console.log('Passable');
} else {
    console.log('Insuffisant');
}

🎯 Opérateurs logiques

let age = 25;
let permis = true;

// AND (&&) - Les deux conditions doivent être vraies
if (age >= 18 && permis) {
    console.log('Vous pouvez conduire');
}

// OR (||) - Au moins une condition doit être vraie
if (age < 18 || !permis) {
    console.log('Vous ne pouvez pas conduire');
}

// NOT (!) - Inverse la condition
let estMajeur = age >= 18;
if (!estMajeur) {
    console.log('Vous êtes mineur');
}

🔄 Switch

let jour = 'lundi';

switch (jour) {
    case 'lundi':
        console.log('Début de semaine');
        break;
    case 'mercredi':
        console.log('Milieu de semaine');
        break;
    case 'vendredi':
        console.log('Bientôt le weekend!');
        break;
    case 'samedi':
    case 'dimanche':
        console.log('Weekend!');
        break;
    default:
        console.log('Jour de semaine');
}

🔁 Boucle For

// Compter de 0 à 4
for (let i = 0; i < 5; i++) {
    console.log(i);
}
// Affiche: 0, 1, 2, 3, 4

// Parcourir un tableau
let fruits = ['pomme', 'banane', 'orange'];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

// For...of (plus moderne)
for (let fruit of fruits) {
    console.log(fruit);
}

🔂 Boucle While

// While - Tant que la condition est vraie
let compteur = 0;
while (compteur < 5) {
    console.log(compteur);
    compteur++;
}

// Do...While - Exécute au moins une fois
let nombre = 10;
do {
    console.log(nombre);
    nombre--;
} while (nombre > 0);

🛑 Break et Continue

// Break - Sortir de la boucle
for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break; // Arrête la boucle
    }
    console.log(i); // 0, 1, 2, 3, 4
}

// Continue - Passer à l'itération suivante
for (let i = 0; i < 5; i++) {
    if (i === 2) {
        continue; // Saute cette itération
    }
    console.log(i); // 0, 1, 3, 4
}

❓ Opérateur ternaire

// Condition ? valeurSiVrai : valeurSiFaux
let age = 20;
let statut = age >= 18 ? 'Majeur' : 'Mineur';
console.log(statut); // 'Majeur'

// Équivalent à:
let statut2;
if (age >= 18) {
    statut2 = 'Majeur';
} else {
    statut2 = 'Mineur';
}
💡 Exemple : Table de multiplication
function tableMultiplication(nombre) {
    console.log(`Table de ${nombre}:`);
    for (let i = 1; i <= 10; i++) {
        console.log(`${nombre} × ${i} = ${nombre * i}`);
    }
}

tableMultiplication(7);

Leçon 5 : Manipulation du DOM

Apprenez à interagir avec le HTML et à modifier vos pages web dynamiquement.

🌳 Qu'est-ce que le DOM ?

Le DOM (Document Object Model) est une représentation de votre page HTML sous forme d'objets JavaScript. Il permet de lire et modifier le contenu, la structure et le style de la page.

🔍 Sélectionner des éléments

// Par ID
let titre = document.getElementById('monTitre');

// Par classe (retourne une collection)
let paragraphes = document.getElementsByClassName('texte');

// Par balise
let tousLesDivs = document.getElementsByTagName('div');

// Sélecteur CSS (le premier élément)
let premier = document.querySelector('.maClasse');
let premierP = document.querySelector('p');

// Tous les éléments (retourne NodeList)
let tous = document.querySelectorAll('.maClasse');
let tousP = document.querySelectorAll('p');

✏️ Modifier le contenu

// HTML
// <h1 id="titre">Ancien titre</h1>
// <p class="texte">Ancien texte</p>

// Modifier le texte
let titre = document.getElementById('titre');
titre.textContent = 'Nouveau titre';

// Modifier le HTML
let texte = document.querySelector('.texte');
texte.innerHTML = '<strong>Texte en gras</strong>';

// Récupérer le contenu
console.log(titre.textContent);
console.log(texte.innerHTML);

🎨 Modifier le style

let element = document.getElementById('monElement');

// Modifier un style
element.style.color = 'red';
element.style.backgroundColor = 'yellow';
element.style.fontSize = '24px';
element.style.padding = '20px';

// Ajouter/Retirer des classes
element.classList.add('active');
element.classList.remove('hidden');
element.classList.toggle('visible'); // Ajoute si absent, retire si présent

// Vérifier si une classe existe
if (element.classList.contains('active')) {
    console.log('Element est actif');
}

➕ Créer et ajouter des éléments

// Créer un nouvel élément
let nouveauDiv = document.createElement('div');
nouveauDiv.textContent = 'Je suis un nouveau div';
nouveauDiv.className = 'maClasse';

// Ajouter à la fin d'un parent
document.body.appendChild(nouveauDiv);

// Insérer avant un élément
let parent = document.getElementById('container');
let reference = document.getElementById('element1');
parent.insertBefore(nouveauDiv, reference);

// Créer et ajouter plusieurs éléments
for (let i = 1; i <= 5; i++) {
    let item = document.createElement('li');
    item.textContent = 'Item ' + i;
    document.getElementById('liste').appendChild(item);
}

🗑️ Supprimer des éléments

// Sélectionner et supprimer
let element = document.getElementById('aSupprimer');
element.remove();

// Supprimer via le parent
let parent = document.getElementById('container');
let enfant = document.getElementById('element');
parent.removeChild(enfant);

// Vider un élément
let container = document.getElementById('container');
container.innerHTML = '';

🔧 Modifier les attributs

let image = document.getElementById('monImage');

// Récupérer un attribut
let src = image.getAttribute('src');

// Modifier un attribut
image.setAttribute('src', 'nouvelle-image.jpg');
image.setAttribute('alt', 'Description');

// Raccourcis pour attributs communs
image.src = 'image.jpg';
image.alt = 'Photo';

let lien = document.getElementById('monLien');
lien.href = 'https://google.com';
lien.target = '_blank';
💡 Exemple complet : Todo List simple
function ajouterTache() {
    // Récupérer la valeur de l'input
    let input = document.getElementById('tacheInput');
    let texte = input.value;
    
    if (texte === '') {
        alert('Veuillez entrer une tâche');
        return;
    }
    
    // Créer un nouvel élément

    li.textContent = texte;
    
    // Ajouter à la liste
    document.getElementById('listeTaches').appendChild(li);
    
    // Vider l'input
    input.value = '';
}

  • Utilisez querySelector/querySelectorAll (plus flexible)
  • Préférez classList à la modification directe du style
  • Stockez les sélections dans des variables si utilisées plusieurs fois
  • Vérifiez que l'élément existe avant de le manipuler

Leçon 6 : Les Événements

Réagissez aux actions de l'utilisateur et rendez vos pages vraiment interactives !

🖱️ Qu'est-ce qu'un événement ?

Un événement est une action qui se produit dans le navigateur : clic, survol, saisie clavier, chargement de page, etc. JavaScript peut "écouter" ces événements et réagir en conséquence.

👆 Événements de souris

let bouton = document.getElementById('monBouton');

// Click (clic simple)
bouton.addEventListener('click', function() {
    console.log('Bouton cliqué!');
});

// Double-click
bouton.addEventListener('dblclick', function() {
    console.log('Double clic!');
});

// Mouseenter (survol)
bouton.addEventListener('mouseenter', function() {
    bouton.style.backgroundColor = 'blue';
});

// Mouseleave (quitter le survol)
bouton.addEventListener('mouseleave', function() {
    bouton.style.backgroundColor = 'gray';
});

// Mousedown et mouseup
bouton.addEventListener('mousedown', function() {
    console.log('Bouton pressé');
});

bouton.addEventListener('mouseup', function() {
    console.log('Bouton relâché');
});

⌨️ Événements de clavier

let input = document.getElementById('monInput');

// Keydown (touche pressée)
input.addEventListener('keydown', function(event) {
    console.log('Touche pressée:', event.key);
});

// Keyup (touche relâchée)
input.addEventListener('keyup', function(event) {
    console.log('Touche relâchée:', event.key);
});

// Détecter une touche spécifique
document.addEventListener('keydown', function(event) {
    if (event.key === 'Enter') {
        console.log('Entrée pressée!');
    }
    if (event.key === 'Escape') {
        console.log('Échap pressée!');
    }
});

📝 Événements de formulaire

let input = document.getElementById('email');
let formulaire = document.getElementById('monForm');

// Input (à chaque changement)
input.addEventListener('input', function(event) {
    console.log('Valeur actuelle:', event.target.value);
});

// Change (après la saisie)
input.addEventListener('change', function() {
    console.log('Valeur finale:', input.value);
});

// Focus (lorsque l'élément reçoit le focus)
input.addEventListener('focus', function() {
    input.style.border = '2px solid blue';
});

// Blur (lorsque l'élément perd le focus)
input.addEventListener('blur', function() {
    input.style.border = '1px solid gray';
});

// Submit (soumission du formulaire)
formulaire.addEventListener('submit', function(event) {
    event.preventDefault(); // Empêche le rechargement
    console.log('Formulaire soumis!');
});

📄 Événements de page

// DOMContentLoaded (DOM chargé)
document.addEventListener('DOMContentLoaded', function() {
    console.log('Page chargée!');
    // Votre code ici
});

// Load (tout chargé : images, CSS, etc.)
window.addEventListener('load', function() {
    console.log('Tout est chargé!');
});

// Scroll
window.addEventListener('scroll', function() {
    console.log('Position:', window.scrollY);
});

// Resize (redimensionnement)
window.addEventListener('resize', function() {
    console.log('Largeur:', window.innerWidth);
});

🎯 L'objet event

bouton.addEventListener('click', function(event) {
    // Informations sur l'événement
    console.log(event.type);        // 'click'
    console.log(event.target);      // Élément cliqué
    console.log(event.clientX);     // Position X de la souris
    console.log(event.clientY);     // Position Y de la souris
    
    // Empêcher le comportement par défaut
    event.preventDefault();
    
    // Arrêter la propagation
    event.stopPropagation();
});

🗑️ Retirer un événement

function maFonction() {
    console.log('Événement déclenché');
}

// Ajouter
bouton.addEventListener('click', maFonction);

// Retirer
bouton.removeEventListener('click', maFonction);

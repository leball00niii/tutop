Leçon 1 : Introduction au CSS

Découvrez le CSS (Cascading Style Sheets) et comment l'utiliser pour donner du style à vos pages HTML.

🎨 Qu'est-ce que le CSS ?

CSS signifie Cascading Style Sheets (Feuilles de Style en Cascade). Le CSS permet de définir l'apparence visuelle de vos pages web : couleurs, polices, espacements, positionnement, animations, etc.

Si le HTML est le squelette d'une maison, le CSS en est la décoration : peinture, meubles, disposition, etc.

📝 Les trois façons d'utiliser le CSS

1. CSS Inline (dans l'élément HTML)
<p style="color: red; font-size: 20px;">Texte rouge</p>

❌ Non recommandé : difficile à maintenir

2. CSS Interne (dans la balise <style>)
<head>
    <style>
        p {
            color: red;
            font-size: 20px;
        }
    </style>
</head>

✅ Bon pour les petits projets

3. CSS Externe (fichier séparé)
<head>
    <link rel="stylesheet" href="style.css">
</head>

✅✅ Recommandé : meilleure organisation et réutilisabilité

🔧 Syntaxe de base du CSS

sélecteur {
    propriété: valeur;
    propriété2: valeur2;
}
💡 Exemple concret :
h1 {
    color: blue;
    font-size: 32px;
    text-align: center;
}
✅ Bonnes pratiques :
  • Utilisez des fichiers CSS externes pour les projets
  • Organisez votre code CSS de manière logique
  • Utilisez l'indentation pour la lisibilité
  • Commentez votre code : /* Ceci est un commentaire */

Leçon 2 : Sélecteurs CSS

Apprenez à cibler précisément les éléments HTML que vous souhaitez styliser.

🎯 Sélecteur d'élément

Cible tous les éléments d'un certain type.

p {
    color: green;
}

h1 {
    font-size: 36px;
}

🏷️ Sélecteur de classe (.)

Cible les éléments avec une classe spécifique.

/* HTML */
<p class="important">Texte important</p>

/* CSS */
.important {
    color: red;
    font-weight: bold;
}

🆔 Sélecteur d'ID (#)

Cible un élément unique avec un ID spécifique.

/* HTML */
<div id="header">En-tête</div>

/* CSS */
#header {
    background-color: navy;
    color: white;
}

🌟 Sélecteur universel (*)

* {
    margin: 0;
    padding: 0;
}

👨‍👩‍👧 Sélecteurs combinés

/* Descendant : tous les p dans div */
div p {
    color: blue;
}

/* Enfant direct */
div > p {
    color: red;
}

/* Plusieurs classes */
.important.urgent {
    background: yellow;
}

/* Groupement */
h1, h2, h3 {
    font-family: Arial;
}
✅ Priorité des sélecteurs :
  • ID (#) : Priorité la plus haute
  • Classe (.) : Priorité moyenne
  • Élément : Priorité la plus basse
  • !important : Force la priorité (à utiliser rarement)
📝 Exercices Pratiques

Exercice 2 :

Exercice 3 :

Leçon 3 : Couleurs & Texte

Apprenez à utiliser les couleurs et à styliser le texte pour rendre vos pages attractives.

🎨 Les couleurs en CSS

/* Noms de couleurs */
color: red;
background-color: blue;

/* Hexadécimal */
color: #ff0000;
background-color: #3498db;

/* RGB */
color: rgb(255, 0, 0);
background-color: rgb(52, 152, 219);

/* RGBA (avec transparence) */
color: rgba(255, 0, 0, 0.5);
background-color: rgba(0, 0, 0, 0.8);

/* HSL */
color: hsl(0, 100%, 50%);

✏️ Propriétés du texte

/* Police */
font-family: Arial, sans-serif;
font-size: 16px;
font-weight: bold; /* ou 100-900 */
font-style: italic;

/* Alignement */
text-align: center; /* left, right, justify */
text-decoration: underline; /* none, overline, line-through */
text-transform: uppercase; /* lowercase, capitalize */

/* Espacement */
line-height: 1.6;
letter-spacing: 2px;
word-spacing: 5px;

/* Ombre */
text-shadow: 2px 2px 4px rgba(0,0,0,0.5);
💡 Exemple complet :
h1 {
    color: #2c3e50;
    font-family: 'Arial', sans-serif;
    font-size: 48px;
    font-weight: bold;
    text-align: center;
    text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
}

🌈 Dégradés (Gradients)

/* Dégradé linéaire */
background: linear-gradient(to right, red, blue);
background: linear-gradient(45deg, #ff6b6b, #4ecdc4);

/* Dégradé radial */
background: radial-gradient(circle, red, blue);
✅ Conseils couleurs :
  • Utilisez une palette de couleurs cohérente
  • Vérifiez le contraste pour la lisibilité
  • RGB avec alpha pour la transparence
  • Outils : Adobe Color, Coolors
📝 Exercices Pratiques
Exercice 1 :

Exercice 2 :

Exercice 3 :

Leçon 4 : Le Box Model (Modèle de boîte)

Comprenez comment fonctionne le modèle de boîte CSS, élément fondamental de la mise en page.

📦 Qu'est-ce que le Box Model ?

Chaque élément HTML est une boîte rectangulaire composée de 4 parties :

  • Content : Le contenu (texte, image, etc.)
  • Padding : Espace entre le contenu et la bordure
  • Border : La bordure autour du padding
  • Margin : Espace extérieur, entre l'élément et les autres

📏 Margin (Marge externe)

/* Toutes les marges */
margin: 20px;

/* Marges individuelles */
margin-top: 10px;
margin-right: 15px;
margin-bottom: 10px;
margin-left: 15px;

/* Raccourci (haut, droite, bas, gauche) */
margin: 10px 20px 10px 20px;

/* Haut/bas et gauche/droite */
margin: 10px 20px;

/* Centrer un élément */
margin: 0 auto;

🛡️ Padding (Marge interne)

/* Tout le padding */
padding: 20px;

/* Paddings individuels */
padding-top: 10px;
padding-right: 15px;
padding-bottom: 10px;
padding-left: 15px;

/* Raccourci */
padding: 10px 20px;

🖼️ Border (Bordure)

/* Bordure complète */
border: 2px solid black;

/* Bordures individuelles */
border-top: 1px dashed red;
border-right: 3px dotted blue;
border-bottom: 2px solid green;
border-left: 1px solid gray;

/* Propriétés séparées */
border-width: 2px;
border-style: solid; /* solid, dashed, dotted, double */
border-color: red;

/* Coins arrondis */
border-radius: 10px;
border-radius: 50%; /* Cercle */

📐 Width et Height

width: 300px;
height: 200px;
max-width: 100%;
min-height: 400px;

/* Box-sizing */
box-sizing: border-box; /* Inclut padding et border dans width/height */
💡 Exemple complet :
.box {
    width: 300px;
    height: 200px;
    padding: 20px;
    border: 3px solid #3498db;
    margin: 20px auto;
    border-radius: 10px;
    box-sizing: border-box;
}
✅ Conseils Box Model :
  • Utilisez box-sizing: border-box pour simplifier les calculs
  • Margin peut être négatif, pas padding
  • Les marges verticales se chevauchent (margin collapse)
  • Utilisez margin: 0 auto pour centrer horizontalement

Exercice 2 :

Leçon 5 : Positionnement

Maîtrisez les différentes méthodes de positionnement pour placer vos éléments exactement où vous le souhaitez.

📍 Position: Static (par défaut)

Position normale dans le flux du document.

position: static; /* Valeur par défaut */

📌 Position: Relative

Positionné par rapport à sa position normale.

.element {
    position: relative;
    top: 20px;    /* Décalage depuis le haut */
    left: 30px;   /* Décalage depuis la gauche */
}

🎯 Position: Absolute

Positionné par rapport au parent positionné le plus proche.

.parent {
    position: relative;
}

.child {
    position: absolute;
    top: 0;
    right: 0;
}

📎 Position: Fixed

Fixé par rapport à la fenêtre du navigateur.

.navbar {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    background: white;
}

🔝 Position: Sticky

Combine relative et fixed selon le scroll.

.header {
    position: sticky;
    top: 0;
    background: white;
}

🎚️ Z-index (superposition)

.element1 {
    position: relative;
    z-index: 1;
}

.element2 {
    position: relative;
    z-index: 10; /* Au-dessus de element1 */
}

🎪 Display

display: block;        /* Prend toute la largeur */
display: inline;       /* Sur la même ligne */
display: inline-block; /* Inline avec width/height */
display: none;         /* Caché */
display: flex;         /* Flexbox */
display: grid;         /* Grid */
💡 Exemple : Badge en absolute
.card {
    position: relative;
    width: 300px;
    height: 200px;
}

.badge {
    position: absolute;
    top: 10px;
    right: 10px;
    background: red;
    color: white;
    padding: 5px 10px;
    border-radius: 20px;
}
⚠️ Attention :

Un élément en position absolute est retiré du flux normal. Les autres éléments agissent comme s'il n'existait pas.

Leçon 6 : Flexbox

Découvrez Flexbox, l'outil moderne pour créer des mises en page flexibles et responsives facilement.

🎯 Qu'est-ce que Flexbox ?

Flexbox (Flexible Box Layout) est un modèle de mise en page qui permet d'organiser facilement des éléments dans un conteneur, même quand leur taille est inconnue ou dynamique.

📦 Activer Flexbox

.container {
    display: flex;
}

➡️ Direction des éléments (flex-direction)

flex-direction: row;            /* Horizontal (défaut) */
flex-direction: row-reverse;    /* Horizontal inversé */
flex-direction: column;         /* Vertical */
flex-direction: column-reverse; /* Vertical inversé */

📐 Alignement horizontal (justify-content)

justify-content: flex-start;    /* À gauche (défaut) */
justify-content: flex-end;      /* À droite */
justify-content: center;        /* Centré */
justify-content: space-between; /* Espace entre */
justify-content: space-around;  /* Espace autour */
justify-content: space-evenly;  /* Espace égal */

📏 Alignement vertical (align-items)

align-items: stretch;     /* Étire (défaut) */
align-items: flex-start;  /* En haut */
align-items: flex-end;    /* En bas */
align-items: center;      /* Centré */
align-items: baseline;    /* Ligne de base */

🔄 Retour à la ligne (flex-wrap)

flex-wrap: nowrap;       /* Pas de retour (défaut) */
flex-wrap: wrap;         /* Retour à la ligne */
flex-wrap: wrap-reverse; /* Retour inversé */

🎛️ Propriétés des éléments enfants

/* Flexibilité */
flex-grow: 1;    /* Grandit pour remplir l'espace */
flex-shrink: 1;  /* Peut rétrécir si nécessaire */
flex-basis: 200px; /* Taille de base */

/* Raccourci */
flex: 1; /* équivalent à flex-grow: 1 */

/* Ordre */
order: 2; /* Change l'ordre d'affichage */

/* Alignement individuel */
align-self: center;
💡 Exemple : Centrer parfaitement
.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}
💡 Exemple : Navbar responsive
.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 20px;
    background: #333;
}

.nav-links {
    display: flex;
    gap: 20px;
}
💡 Exemple : Grille de cartes
.cards-container {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
}

.card {
    flex: 1 1 300px; /* Grandit, rétrécit, base 300px */
    min-width: 250px;
}
✅ Astuces Flexbox :
  • Utilisez gap pour espacer les éléments facilement
  • flex: 1 fait grandir un élément pour remplir l'espace
  • Combinez justify-content et align-items pour centrer
  • flex-wrap: wrap pour rendre responsive
